# Experience Cloud Audiences

[!DNL Adobe Experience Cloud audiences] können [!DNL Experience Cloud] Anwendungen Informationen über Website-Besucher kommunizieren und freigeben, sodass von jeder Anwendung dieselben Daten verwendet werden können.

Weitere Informationen zu [Experience Cloud-Zielgruppen](https://experienceleague.adobe.com/docs/core-services/interface/audiences/audience-library.html?lang=de) (Zielgruppenbibliothek) finden Sie im *Handbuch zur Oberfläche und Administration von Experience Cloud*.
