# Inhalte suchen und filtern

Suchen nach Assets anhand von Keywords in der [!UICONTROL Angebote]-Bibliothek in [!DNL Adobe Target].

## Nach Code-Angeboten nach Keyword suchen

1. Klicken Sie auf **[!UICONTROL Angebote]** > **[!UICONTROL Angebote codieren]**.
1. Geben Sie die gewünschten Keywords in das Feld **[!UICONTROL Suche in]** ein.

   Sie können die Dropdown-Liste verwenden, um nach [!UICONTROL Angebotsname] [!UICONTROL AEM-Pfaden] oder [!UICONTROL AEM-Tags] zu filtern.

   Sie können auch mithilfe der Optionen [!UICONTROL Typ], [!UICONTROL Source] und [!UICONTROL AEM-Typ] filtern, indem Sie auf das Symbol [!UICONTROL Filter anzeigen] ![Symbol „Filter anzeigen“](/help/main/assets/icons/Filter.svg) klicken, um Ihre Suche weiter zu verfeinern.

## Bildangebote nach Keyword suchen {#section_2465A71BC95942588F586B1EC8B9E5DB}

1. Klicken Sie **[!UICONTROL Angebote]** > **[!UICONTROL Bildangebote]**.

1. (Optional) Wechseln Sie zwischen [!UICONTROL Kartenansicht] und [!UICONTROL Listenansicht] und klicken Sie auf das [!UICONTROL Kartenansicht]-Symbol ( ![Kartenansichtssymbol](/help/main/assets/icons/ViewCard.svg) ) oder das [!UICONTROL Listenansicht]-Symbol ( ![Listenansicht-Symbol](/help/main/assets/icons/ViewList.svg) ) in der oberen rechten Ecke der [!UICONTROL Asset]-Bibliothek.
1. Klicken Sie auf **[!UICONTROL Nur Inhalt]**-Symbol ( ![Nur Inhalt](/help/main/assets/icons/RailLeft.svg)-Symbol) in der oberen linken Ecke, um das Suchfeld anzuzeigen.
1. Geben Sie im Suchfeld einen Suchbegriff für die Assets ein, die Sie suchen möchten, und drücken Sie dann die [!UICONTROL Eingabetaste].
1. (Bedingt) Verwenden Sie den Bereich **[!UICONTROL Filter und Optionen]** auf der linken Seite, um Ihre Suche zu verfeinern.

   Sie können nach folgenden Kriterien filtern:

   * **[!UICONTROL Dateityp]**:
      * [!UICONTROL Bilder]
      * [!UICONTROL Dokumente]
      * [!UICONTROL Multimedia]
      * [!UICONTROL Archive]
   * **[!UICONTROL Dateigröße]**: Verwenden Sie den Schieberegler, um die gewünschte Dateigröße auszuwählen: [!UICONTROL Kleinste], [!UICONTROL Klein], [!UICONTROL Medium], [!UICONTROL Groß] oder [!UICONTROL Größte].
   * **[!UICONTROL Zuletzt geändert]**: Verwenden Sie den Schieberegler, um den Zeitraum auszuwählen: [!UICONTROL Zuletzt &#x200B;], [!UICONTROL Stunde], [!UICONTROL Tag], [!UICONTROL Woche], [!UICONTROL Monat], [!UICONTROL Jahr] oder [!UICONTROL Alle Assets].
   * **[!UICONTROL Genehmigungsstatus]**: [!UICONTROL Genehmigt] oder [!UICONTROL Abgelehnt]
   * **[!UICONTROL Ablaufstatus]**: [!UICONTROL Expired].
   * **[!UICONTROL Durchschnittsbewertung]**: 1 bis 5.
   * **[!UICONTROL Ausrichtung]**: [!UICONTROL Horizontal], [!UICONTROL Vertikal] oder [!UICONTROL Quadrat].
   * **[!UICONTROL Style]**: [!UICONTROL Color] oder [!UICONTROL Black &amp; White].
   * **[!UICONTROL Nutzungsbewertung]**: [!UICONTROL Minimum] bis [!UICONTROL Maximum].
   * **[!UICONTROL Impression Score]**: [!UICONTROL Minimum] bis [!UICONTROL Maximum].
   * **[!UICONTROL Klicken Sie auf Score]**: [!UICONTROL Minimum] bis [!UICONTROL Maximum].
