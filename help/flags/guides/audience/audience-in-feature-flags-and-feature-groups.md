# Zielgruppe in Feature Flags und Feature Groups {#audience-overview}

Flags bietet Zielgruppen-Targeting für Feature Flags und Feature Groups. Sie können Kontextvariablen kombinieren, um genau zu steuern, welche Benutzer eine Funktion erhalten.

Informationen zu kontextbasiertem Targeting (z. B. Targeting nach der aktiven Sprache oder dem Gerätetyp des Benutzers) finden Sie unter [Verwenden des Kontexts in Zielgruppenregeln](using-context-in-audience-rules.md).
